# 读者参与度检测（单主角网文默认规则）

> 本文档是 `default-templates/` 中的预置规则，适用于标准单主角网文。
> 当理解层判断项目属于此类型时，可直接复制到 `generated-rules/` 使用。
> 也可作为参考模板，由理解层修改后生成项目专属版本。

## 检测目标

检测章节的读者参与度——钩子强度、微回报密度、爽感交付、是否有足够理由让读者点击下一章。

## 钩子类型

| 类型 | 定义 | 强度判定 |
|------|------|---------|
| crisis | 危机逼近 | 强：生死；中：利益；弱：潜在 |
| mystery | 信息缺口 | 强：核心谜题；中：支线；弱：细节 |
| emotion | 强烈情绪触发 | 强：关系破裂；中：冲突升级；弱：微妙变化 |
| choice | 两难抉择 | 强：不可逆选择；中：重要选择；弱：日常选择 |
| desire | 期待正面结果 | 强：里程碑突破；中：阶段目标；弱：日常期待 |

## 回报类型

| 类型 | 定义 |
|------|------|
| info_reveal | 读者/角色获知新信息 |
| power_display | 角色展示实力 |
| relationship | 关系推进 |
| resource | 资源获取 |
| status | 地位/认可提升 |
| emotional_release | 情感释放 |
| plot_advance | 剧情线推进 |

## 爽感模式（8种标准网文模式）

### 1. 装逼打脸
**结构**: 铺垫(被看不起) → 反转(一招制敌) → 反应(围观者震惊)
**节奏**: 30% 铺垫 → 40% 交付 → 30% 余韵

### 2. 扮猪吃虎
**结构**: 隐藏实力 → 被轻视 → 碾压展示

### 3. 越级反杀
**结构**: 实力差距展示 → 策略/爆发 → 逆转

### 4. 打脸权威
**结构**: 权威出场 → 挑战 → 权威认输

### 5. 反派翻车
**结构**: 反派得意 → 主角反制 → 反派崩盘

### 6. 甜蜜超预期
**结构**: 建立预期 → 超额交付 → 情绪升华

### 7. 迪化误解
**结构**: 随意行为 → 配角过度解读 → 读者优越感

### 8. 身份掉马
**结构**: 长期隐藏 → 触发暴露 → 群体反应

## 硬约束

### HARD-001 [critical]
**条件**: 读者无法理解本章发生了什么
**判定**: 是否存在至少一个可概括的核心事件
**失败处理**: 必须重写

### HARD-002 [critical]
**条件**: 上一章显性承诺零回应
**判定**: 对照上一章钩子，本章是否有任何回应
**失败处理**: 必须补充回应

### HARD-003 [critical]
**条件**: 连续章节完全无推进
**判定**: 状态变更为空且无新信息
**失败处理**: 必须增加推进

### HARD-004 [high]
**条件**: 完全缺乏问题、目标或stakes
**判定**: 章节中是否存在至少一个未解决的问题或目标
**失败处理**: 必须建立stakes

## 软约束

### SOFT-001 [权重: 15]
**条件**: 章末应有至少一个未解决问题或期待锚点
**豁免条件**: 卷末总结章

### SOFT-002 [权重: 10]
**条件**: 不应连续3章使用相同钩子类型
**豁免条件**: 叙事结构要求
**豁免债务**: 下一章必须切换类型

### SOFT-003 [权重: 10]
**条件**: 每章至少1个微回报
**豁免条件**: 纯铺垫章（过渡型）
**豁免债务**: 2章内必须交付回报

### SOFT-004 [权重: 15]
**条件**: 每5章至少1个完整爽感模式交付
**豁免条件**: 文风渐变期

### SOFT-005 [权重: 10]
**条件**: 单一爽感模式不超过总量的50%
**豁免条件**: 无

### SOFT-006 [权重: 10]
**条件**: 开头不应连续3章使用相同模式
**豁免条件**: 无

### SOFT-007 [权重: 15]
**条件**: 新期待不超过每章2个
**豁免条件**: 无

### SOFT-008 [权重: 15]
**条件**: 钩子强度应匹配章节定位（高潮章≥medium，过渡章≥weak）
**豁免条件**: 无

## 评分算法

**基础分**: 100
**扣分方式**: 每条软约束违反扣除权重值
**最终分**: max(0, 100 - 总扣分)

**等级**:
- 85+: PASS
- 70-84: PASS_WITH_WARNINGS
- 50-69: CONDITIONAL
- <50: FAIL
